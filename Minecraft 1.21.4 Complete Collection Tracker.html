  1. Copy the data: Click "Copy CSV to Clipboard" above
  2. Open Google Sheets: Go to sheets.google.com
  3. Create new sheet: Click "Blank" to create a new spreadsheet
  4. Paste data: Click cell A1 and paste (Ctrl+V or Cmd+V)
  5. Format data: Select all data, then go to Data → Split text to columns
  6. Add checkboxes: Select column A, then Insert → Checkbox

Note: Progress is saved only during this browser session. Export to Google Sheets to keep permanent records!
